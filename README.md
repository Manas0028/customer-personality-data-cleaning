# Data Cleaning and Preprocessing using Python (Pandas)

## 📌 Project Overview

This project demonstrates the process of cleaning and preprocessing a real-world dataset using Python and Pandas. The goal was to identify and fix common data quality issues to prepare the dataset for further analysis and visualization.

The dataset used is **Customer Personality Analysis (Marketing Campaign)**, which contains customer demographic information, purchasing behavior, and marketing campaign responses.

---

## 🎯 Objective

* Handle missing values
* Remove duplicate records
* Standardize text formatting
* Convert date columns to the correct format
* Rename column headers for consistency
* Verify and correct data types
* Create a clean dataset ready for analysis

---

## 🧹 Data Cleaning Steps

* Loaded the dataset using Pandas.
* Examined the dataset structure and data types.
* Identified missing values using `isnull()`.
* Filled missing values in the `Income` column using the median.
* Checked and removed duplicate records (if present).
* Renamed column headers to lowercase with underscores.
* Converted the `Dt_Customer` column to datetime format.
* Standardized text values in categorical columns.
* Verified data types and data quality.
* Exported the cleaned dataset as a new CSV file.
